# V

Save time and keep focus with fuzzy searching files and opening them in your
configured `$EDITOR`.

> Default editor is `vi`

## Install

Place in `.bashrc`:

```sh
source v
export EDITOR='vim' # Configure your favorite editor.
```

```sh
$ v bashrc # Will open the bashrc file in vim.
```
